# 🔬 Lab Resource Utilization Platform

> A full-stack laboratory resource management platform that enables research institutions, universities, and laboratories to efficiently manage, schedule, share, and optimize laboratory equipment through intelligent booking, utilization monitoring, and analytics.

---

## 📖 Overview

The **Lab Resource Utilization Platform** is a full-stack web application developed using **React.js** and **Spring Boot**. It enables institutions to efficiently manage laboratory resources by providing equipment inventory management, booking and scheduling, utilization monitoring, maintenance tracking, inter-institution resource sharing, analytics dashboards, and secure user authentication. The platform helps improve equipment utilization, reduce operational costs, and support data-driven decision-making.

---

## ✨ Features

- 🔐 JWT Authentication & OAuth2 Login

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/Apurvasri25/Resource-Usage-Hub.git
```

### Navigate to the Project

```bash
cd Resource-Usage-Hub
```

### Install Frontend Dependencies

```bash
npm install
```

### Start Frontend

```bash
npm run dev
```

### Start Backend

```bash
mvn spring-boot:run
```
